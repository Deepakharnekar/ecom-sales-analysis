# 🛒 Retail Sales Analysis & Dashboard (Superstore Dataset)

## 📌 Project Overview
This project analyzes 51,000+ retail transactions from the Superstore dataset to uncover sales trends, profitability drivers, customer behavior, and operational insights.

The project includes:
- Data Cleaning & Exploratory Data Analysis (Python)
- Business Insights & Recommendations
- Interactive Dashboard (Power BI)

---

## 🛠 Tools & Technologies Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (Data Visualization)
- Jupyter Notebook
- Power BI (Dashboard Development)

---

## 📊 Key Business Questions
- Which region generates the highest sales and profit?
- Which product categories drive revenue?
- How do discounts impact profitability?
- What are the monthly and yearly sales trends?
- Which shipping mode is most preferred?

---

## 📈 Key KPIs
- **Total Sales:** 12.6M
- **Total Profit:** 1.47M
- **CAGR:** 23.5%
- **Late Deliveries:** 28.6%
- **Avg Shipping Cost:** 26.37

---

## 🔍 Major Insights
- Discounts above 20% consistently resulted in losses.
- APAC region dominated total sales.
- Canada showed high growth potential.
- Standard Class shipping accounted for 60% of orders.
- Office Supplies drove high transaction volume.
- Technology category generated highest profit margins.
