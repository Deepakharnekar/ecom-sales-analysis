# 🛒 E-Commerce Sales Analysis Project

## 📌 Project Overview
This project analyzes e-commerce sales data using Python.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Insights
- Total Sales Analysis
- Monthly Sales Trend
- Top Selling Products
- Category-wise Performance
- Profit Analysis
